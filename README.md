### 설치

1. [python3.6](https://www.python.org/downloads/) 이상이 설치되어 있어야합니다.
2. 가상환경 생성 및 활성화
```
$ python3 -m venv venv
$ source venv/bin/activate
```

3. requirements 설치
```
$ pip3 install -r requirements.txt
```
4. 실행   
```
$ python3 manage.py runserver 0.0.0.0:8000
```
